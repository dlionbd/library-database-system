# Assignment notes

The original project was prepared as a university database assignment in Azure Data Studio / MS SQL Server.

Covered topics:
- Creating a database with separate data and log files
- Connecting Azure Data Studio to a local MS SQL Server engine
- Designing a relational schema for a library borrowing process
- Creating tables, primary keys, foreign keys, constraints and indexes
- Writing business-oriented SQL queries
- Explaining the `.ipynb` format as a JSON-based text file

Database connectors mentioned in the project:
- MSSQL
- ODBC
- JDBC
